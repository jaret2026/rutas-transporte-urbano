# 🚌 Sistema de Gestión de Rutas de Transporte Urbano
**Universidad Estatal de Milagro - UNEMI**
Asignatura: Introducción a la Ingeniería de Software

## 📋 Descripción
Plataforma web para la gestión, monitoreo y planificación de rutas de transporte público.
- Visualización de recorrido
- Gestión de paradas y horarios
- Reportes operativos
